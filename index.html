Como usar

Formato do Arquivo

O arquivo (.xlsx ou .csv) deve conter as seguintes colunas:

Coluna Tipo Valores aceitos
sub_finalizadora_key INTEGER Código numérico da subfinalizadora
codigo_autorizadora
ou codigo_administradora		 STRING Código da administradora/autorizadora
codigo_bandeira INTEGER Código numérico da bandeira
tipo_tef_key INTEGER 1 = SITEF  |  2 = POS
tipo_cartao STRING Texto contendo: debito, credito, voucher ou parcelado

Sem acento, sem maiúsculas obrigatórias. Ex: "Cartão de Crédito" → CARTAO_CREDITO.

Permissão necessária

Verifique se o seu usuário no ERP possui a permissão abaixo antes de usar a integração:

4302 - API Mapa de Subfinalizadora (Incluir)

Tenant e token de acesso

O Tenant é o identificador do cliente no Bluesoft ERP. Na URL erp.bluesoft.com.br/minhaempresa, o tenant é minhaempresa.

Para gerar o Token:

  1. Faça login no ERP Bluesoft com seu usuário.
  2. Clique no nome do seu usuário (canto superior direito) e selecione Editar Perfil.
  3. Role a página até a seção Token para API e clique em Gerar token.
  4. Copie o token gerado e cole no campo Token nas Configurações acima.

Como usar

Permissão necessária

Verifique se o seu usuário no ERP possui a permissão abaixo antes de usar a integração:

4302 - API Mapa de Subfinalizadora (Incluir)

Gerando o token de acesso

Siga estes passos para gerar o token e usá-lo no campo de autenticação:

  1. Faça login no ERP Bluesoft com seu usuário.
  2. Clique no seu nome (canto superior direito) e selecione Editar Perfil.
  3. Role a página até o final e clique em Gerar token.
  4. Copie o token gerado.
  5. Cole o token no campo Token na seção Configurações acima.

Upload de Arquivo

  1. Configure o Tenant e Token nas configurações acima
  2. Selecione um arquivo Excel (.xlsx) ou CSV
  3. Para CSV, selecione a codificação correta se houver problemas com acentuação
  4. Visualize os dados antes do envio
  5. Clique em "Enviar para API" para processar
  6. Acompanhe o progresso e verifique os resultados

Envio Manual

  1. Preencha todos os campos do formulário
  2. Clique em "Enviar Dados"
  3. Verifique o status na seção de resultados

Formato do Arquivo

O arquivo deve conter as seguintes colunas:

  • sub_finalizadora_key (INTEGER): Código da subfinalizadora
  • codigo_autorizadora ou codigo_administradora (STRING): Código da administradora/autorizadora
  • codigo_bandeira (INTEGER): Código da bandeira
  • tipo_tef_key (INTEGER): 1 = SITEF, 2 = POS
  • tipo_cartao (STRING): Cartão de Débito, Cartão de Crédito, Cartão Voucher, Cartão Parcelado
